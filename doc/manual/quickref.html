1.1.5

Plugins

Escape

Stop current plugin operation

F1

Start Pick

Stellar evaluation

If there are one or more plugins active, additional mouse or keyboard bindings may be present. In
general, the left mouse button is used to select, pick or move, and the right mouse button is used to
draw a shape for the operation.
